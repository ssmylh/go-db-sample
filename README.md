# go-db-sample

Go database samples.

## Dependency

- For all sample

```
go get github.com/mattn/go-sqlite3
```

- For genmai sample

```
go get github.com/naoina/genmai
```

- For gorp sample

```
go get gopkg.in/gorp.v1
```

- For argen sample

```
go get -u github.com/monochromegane/argen/...
```
